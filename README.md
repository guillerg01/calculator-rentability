# Rentability Calculator

Real-time investment **rentability calculator** with interactive charts — built with
**Next.js, NextUI and Recharts**. Enter parameters, get instant projections and validations.

## Stack

- **Next.js** + **TypeScript**
- **NextUI** + **Tailwind CSS**
- **Recharts** for data visualization
- **Framer Motion** for animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- Real-time recalculation as inputs change
- Multiple strategy comparison
- Interactive charts (Recharts)
- Input validation and responsive UI
